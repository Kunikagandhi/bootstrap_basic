About Bootstrap:

Bootstrap has become an essential a tool for front-end developers. The Web app development toolkit was created by former Twitter employees Mark Otto and Jacob Thornton. Bootstrap is a giant collection of handy, reusable bits of code written in HTML, CSS, and JavaScript. It is also a front-end development framework that enables developers & designers to quickly build fully responsive websites. Bootstrap saves you from writing lots of CSS code, giving you more time to spend on designing webpages. Now, you can get straight to filling your containers with content. Bootstrap comes with its own code for automatically resizing images based on the current screen size. Just add the .img-responsive class to your images, and the predefined CSS rules take care of the rest.
As with so many open-source projects, Bootstrap has a large community of designers and developers behind it. Being hosted on GitHub makes it easy for developers to modify and contribute to its codebase. It also makes it easy for people to collaborate, lend their advice, and interact with peers and fellow users. As the popularity of Bootstrap grew, people started creating templates based on Bootstrap in order to accelerate the web development process even further.
